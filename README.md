# LR Engenharia e Segurança do Trabalho

Site institucional desenvolvido para a empresa **LR Engenharia e Segurança do Trabalho**, com foco em apresentação corporativa, identidade visual moderna e divulgação de serviços técnicos relacionados à engenharia, segurança do trabalho, assessoria e operações em campo.

---

## Sobre o Projeto

O projeto foi desenvolvido com o objetivo de criar uma presença digital moderna, profissional e responsiva para a empresa, valorizando:

* identidade visual corporativa;
* apresentação dos serviços;
* exibição de obras e operações;
* experiência visual premium;
* organização estrutural;
* navegação intuitiva;
* contato rápido via WhatsApp.

O site possui arquitetura multipágina e foi desenvolvido utilizando HTML, CSS e JavaScript puro, priorizando performance, organização e manutenção simplificada.
